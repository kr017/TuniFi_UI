## Description

This project is clone of Google Keep App.

### Features

- SignIn
- SignUp
- Dark / Light Mode
- Create Note
- Update Note
- Pin / Unpin Note
- Archive / Unarchive Note
- List View
- Grid View
- Delete Note

## Tech Stack

- React
- Express JS
- Mongoose
- MongoDB
